# Company Credit Profiling and Qualitative Risk Assessment System

An AI-powered web application developed to automate company credit profiling and qualitative risk assessment using Machine Learning.

## Features
- Company & promoter profiling
- AI-based risk prediction using Random Forest
- Credit score generation (300–900)
- Loan recommendation system
- Risk analytics dashboard
- Automated PDF report generation

## Technologies Used
- HTML, CSS, Bootstrap
- Python, Flask
- Scikit-learn, Pandas, NumPy
- Chart.js, ReportLab

## Purpose
The system helps financial institutions analyze company credibility, predict risk levels, and make faster lending decisions through automated credit assessment.
